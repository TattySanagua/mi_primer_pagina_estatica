¿Qué es GitHub?

GitHub es una plataforma en la nube que permite almacenar, organizar y compartir proyectos web. Funciona como una especie de repositorio donde guardamos nuestros archivos online, en lugar de tenerlos solo en nuestra computadora.

Una de sus principales ventajas es que permite trabajar de forma colaborativa y llevar control de versiones, lo que significa que podemos guardar cambios, volver atrás o trabajar con otras personas sin perder información.

¿Qué es GitHub Pages?

GitHub Pages es un servicio gratuito que permite publicar sitios web estáticos directamente desde un repositorio de GitHub. Con solo subir nuestros archivos HTML y CSS ya podemos tener una página disponible en Internet.

Lo más interesante es que no necesitamos pagar hosting ni configurar servidores: GitHub Pages se encarga de todo. Simplemente activamos la opción desde las configuraciones del repositorio y obtenemos una URL pública.
